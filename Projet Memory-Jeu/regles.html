Règles du jeu

Comment jouer à ce jeu de mémoire:

Dans ce jeu de mémoire, des cartes sont disposées en colonnes et rangées. Vous pouvez changer le nombre de mots affichés dans le jeu, et ainsi le niveau de difficulté, grâce aux boutons situés en-dessous du jeu.

Vous devez retrouver les paires de mots en les retournant 2 par 2. Si les mots sont identiques, vous avez trouvé la paire et vous avez gagné, sinon recommencez.

Chaque fois que vous commencez un nouveau jeu, une sélection aléatoire des mots vous garantit un jeu différent à chaque fois, vous pouvez donc rejouer autant de fois que vous le souhaitez.
